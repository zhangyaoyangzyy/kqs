## 使用说明

### 一. 脚本密钥激活

1. 初次购买脚本需要进行密钥激活。
2. 购买密钥请向您的购买方索取。
3. 此脚本仅限一台电脑使用，更换电脑需要重新购买密钥。

### 二. IP 配置

1. 根据配置表的要求对应行列进行配置。
2. 请提前测试配置的 IP 是否有效。
3. 为防止“女巫攻击”，我们强烈建议 **1 IP 对应 1 钱包**。

### 三. 钱包配置

1. 根据配置表的要求对应行列进行配置。
2. 请注意区分钱包地址与私钥。
3. 强烈推荐每个项目使用独立的钱包。
4. **使用重新注册的新钱包**！
5. 钱包一旦导入将消耗钱包的使用次数，删除后无法找回，消耗的钱包数也不予返还。

### 四. Telegram 养号配置

1. **Telegram** 一定要三端登录（手机、电脑网页（ADS）和脚本客户端网页）。
2. 注意使用脚本的养号功能，**每天点击“一键养号”**。
